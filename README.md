Para realizar el Fabuloso Sistema de Gestión de Pedidos y Productos
trabajé con dos productos utilizano un producto base abstracto. La clase abstracta
Producto implementa una interfaz llamada Listable que tiene un método que lista las 
características del producto. Trabajé con dos productos que heredan la clase abstracta Producto:
Gaseosa y Vino (que no falte nunca)

Para los pedidos definí una clase Pedido que tiene su ID y un ArrayList de ItemPedido. 
Esta clase tiene como atributos un ProductoPedido, cantidad y total.

Creé dos clases de servicios para gestionar tanto productos como pedidos. Cada producto
y cada pedido tiene un ID que se genera al momento de instanciarlo. Ambas clases tienen
ArrayList donde se almancenan tanto los productos como los pedidos.

Al iniciar el Fabuloso Sistema de Gestión de Pedidos y Productos crea 5 productos
para poder trabajar con el mismo.
